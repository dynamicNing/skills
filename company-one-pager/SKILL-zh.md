# 公司一页纸生成器

将任意公司的核心信息浓缩为一页纸，输出信息密度高、结构清晰的标准化概况，适用于投资筛选、尽调、商务研究。

## 输入处理

接受以下任意输入，无需让用户澄清格式：

| 输入类型 | 处理方式 |
|---|---|
| 公司名称（如"宁德时代"） | 联网搜索最新数据 |
| 股票代码（如"300750.SZ"） | 搜索财务数据源 |
| URL（年报、IR页面） | 抓取并提取关键数据 |
| 粘贴文本 / 上传文件 | 直接解析 |
| 模糊描述（"那个做电池的"） | 推断，必要时确认 |

## 输出格式

默认输出 **Markdown** 格式。如用户要求飞书文档、HTML 等，自动适配。

严格使用以下结构——除非数据确实不可获取，每个板块均为必填：

```
# [公司名称] ([代码]) — 一页纸
> 生成日期：[日期] | 数据截至：[最新可用日期]
```

## 数据采集策略

### 第一步：收集核心数据

通过联网搜索获取：
1. 公司官网 / 投资者关系页面
2. 最新财务数据（营收、利润、利润率）
3. 近期新闻和分析师观点
4. 行业报告和竞争格局

A 股公司优先数据源：
- 东方财富 / 同花顺 — 财务数据
- 巨潮资讯 — 公告原文
- Wind / iFinD — 行业数据

美股 / 港股：
- Yahoo Finance、SEC EDGAR、港交所
- Bloomberg、Seeking Alpha

### 第二步：交叉验证

关键财务数据至少从2个来源交叉核实。置信度低的数据用 `[估]` 标记。

### 第三步：综合成文

填充每个板块。当精确数据不可得时：
- 使用最近可用数据并标注日期
- 使用估计值并标注 `[估]`
- 仅在万不得已时写"数据未公开披露"

## 质量标准

1. **信息密度** — 每个字都有价值，不写废话
2. **时效性** — 必须标注财务数据日期，优先使用TTM
3. **客观性** — 多空两面公平呈现，不做啦啦队
4. **具体性** — "毛利率35%，同比下降3pct" 胜过 "利润率在下降"
5. **完整性** — 所有板块必填，缺失数据要显式标注

## 多公司模式

当用户提供多家公司时，逐一生成一页纸，最后附加对比汇总表，涵盖：市值、营收、利润率、增长率、PE。
